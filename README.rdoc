== KANJI DRILL APP

Simple app I made when I was going to take the Japanese language proficiency test(Noryoku Shikken).
App provides user with 2 way of studying - translate from Russian to Japanese and from Japanese to Russian. 
Simple design represents drills as radiobuttons sets. 
Kanji drills are situated in seeds.rb. 

Main features: 

* Japanese-Russian drills

* Russian-Japanese drills

* Deploy to heroku
